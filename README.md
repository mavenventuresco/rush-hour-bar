# 🍸 Rush Hour Bar

A browser-based bartending time-management game inspired by Bar Rush Unlimited.

Serve customers, mix cocktails, earn tips, and build combos in this endless bar rush!

## How to Play

1. **Pick a glass** from the rack at the top
2. **Add ingredients** from the stock shelves and workstation
3. **Mix/shake** if the recipe requires it
4. **Drag the finished drink** to the customer who ordered it
5. Serve fast for bigger tips — chain correct drinks for combo bonuses!

## Features

- 🍹 24 drink recipes across 6 glass types
- 🍾 46 ingredients across 9 categories
- 🪩 Nightclub bar atmosphere
- 🔥 Combo multiplier system (up to x4)
- 💾 Save and return capability
- 🎵 Original jazz-style soundtrack
- 📋 In-game cocktail menu reference

## Play Online

Just open `index.html` in any browser — no install needed!
